# **App Name**: LegalSphere

## Core Features:

- User Authentication: Secure user authentication with email and password login.
- User Registration: User registration with email and password, with account creation functionality.
- User Dashboard: Dashboard interface with a chat input field for user queries.
- Legal Assistance Chatbot: Chatbot integration to answer user queries related to Indian law. LLM acts as a tool, choosing when/if to provide specific information to the user.
- Prompt Limit: Daily prompt limit enforced (7 free prompts).
- Premium Access: Implement hard coded premium user with unlimited prompt access. Should authenticate a user by email/password combo, then toggle prompt limits off.
- Status Indicator: Display remaining free prompts or premium unlocked status clearly within the UI.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and stability.
- Background color: Light blue (#E8EAF6), near-white for a clean and professional feel.
- Accent color: Teal (#009688), providing contrast without disrupting the serious tone.
- Font: 'Inter', a grotesque-style sans-serif, for both headlines and body text.
- Use formal, law-related icons for navigation (e.g., scales, gavel).
- Clean and organized layout for easy navigation.
- Subtle transitions and loading animations for a polished user experience.
